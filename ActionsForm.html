Understanding JavaScript Alerts

Alert: Simple message box with OK button. Used for notifications.
Confirm: Message box with OK and Cancel buttons. Used for confirmations.
Prompt: Message box with input field and OK/Cancel buttons. Used for user input.

Alert Handling Examples

Example 1 - Handle Alert:
Alert alert = driver.switchTo().alert();
String alertText = alert.getText();
alert.accept(); // or alert.dismiss();
Example 2 - Handle Confirm:
Alert confirm = driver.switchTo().alert();
String confirmText = confirm.getText();
confirm.accept(); // OK button
// or confirm.dismiss(); // Cancel button
Example 3 - Handle Prompt:
Alert prompt = driver.switchTo().alert();
prompt.sendKeys("User input text");
prompt.accept();
Example 4 - Wait for Alert:
WebDriverWait wait = new WebDriverWait(driver, Duration.ofSeconds(10));
Alert alert = wait.until(ExpectedConditions.alertIsPresent());
alert.accept();
