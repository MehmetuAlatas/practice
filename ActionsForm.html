JavaScript Alerts Practice with Selenium Test Examples

Understanding JavaScript Alerts

Alert: Simple message box with OK button. Used for notifications.
Confirm: Message box with OK and Cancel buttons. Used for confirmations.
Prompt: Message box with input field and OK/Cancel buttons. Used for user input.

Basic JavaScript Alerts

Click the buttons below to trigger different types of JavaScript alerts.

Test Case: Basic JavaScript Alerts

Test ID: TC_ALERT_001

Objective: Verify different types of JavaScript alerts can be handled correctly

Test Steps:

  1. Click "Simple Alert" button and verify alert text: "This is a simple alert message!"
  2. Accept the simple alert and verify it's dismissed
  3. Click "Warning Alert" button and verify alert text: "Warning: This action cannot be undone!"
  4. Accept the warning alert and verify it's dismissed
  5. Click "Success Alert" button and verify alert text: "Success: Operation completed successfully!"
  6. Accept the success alert and verify it's dismissed
  7. Click "Error Alert" button and verify alert text: "Error: Something went wrong!"
  8. Accept the error alert and verify result log contains all entries

Expected Results: All alert types should display correct messages, be acceptible, and log entries should be created

Test Data: N/A (alerts use predefined messages)

Selenium Test Code - Basic Alerts

@Test
public void basicAlertsTest() throws InterruptedException {
    // Finding alert buttons
    WebElement simpleAlertButton = driver.findElement(By.xpath("//button[@onclick='showSimpleAlert()']"));
    WebElement warningAlertButton = driver.findElement(By.xpath("//button[@onclick='showWarningAlert()']"));
    WebElement successAlertButton = driver.findElement(By.xpath("//button[@onclick='showSuccessAlert()']"));
    WebElement errorAlertButton = driver.findElement(By.xpath("//button[@onclick='showErrorAlert()']"));
    WebElement alertResult = driver.findElement(By.id("alert-result"));
    // Test Simple Alert
    simpleAlertButton.click();
    
    // Wait for alert to be present
    WebDriverWait wait = new WebDriverWait(driver, Duration.ofSeconds(10));
    Alert simpleAlert = wait.until(ExpectedConditions.alertIsPresent());
    
    // Verify alert text and accept
    String simpleAlertText = simpleAlert.getText();
    Assertions.assertEquals("This is a simple alert message!", simpleAlertText);
    simpleAlert.accept();
    // Test Warning Alert
    warningAlertButton.click();
    Alert warningAlert = wait.until(ExpectedConditions.alertIsPresent());
    String warningAlertText = warningAlert.getText();
    Assertions.assertEquals("Warning: This action cannot be undone!", warningAlertText);
    warningAlert.accept();
    // Test Success Alert
    successAlertButton.click();
    Alert successAlert = wait.until(ExpectedConditions.alertIsPresent());
    String successAlertText = successAlert.getText();
    Assertions.assertEquals("Success: Operation completed successfully!", successAlertText);
    successAlert.accept();
    // Test Error Alert
    errorAlertButton.click();
    Alert errorAlert = wait.until(ExpectedConditions.alertIsPresent());
    String errorAlertText = errorAlert.getText();
    Assertions.assertEquals("Error: Something went wrong!", errorAlertText);
    errorAlert.accept();
    // Verify result log contains entries
    Thread.sleep(1000);
    String resultText = alertResult.getText();
    Assertions.assertTrue(resultText.contains("Simple alert displayed and accepted"));
    Assertions.assertTrue(resultText.contains("Warning alert displayed and accepted"));
}

Confirm Dialogs

These dialogs require user confirmation before proceeding with an action.

Test Case: Confirm Dialogs

Test ID: TC_CONFIRM_001

Objective: Verify confirm dialogs can be accepted or dismissed and proper actions are taken

Test Steps:

  1. Click "Delete Item" button and verify confirm dialog text
  2. Accept the delete confirmation and verify "Delete confirmed" message appears
  3. Click "Leave Page" button and verify confirm dialog about unsaved changes
  4. Dismiss the navigation confirmation and verify "Navigation cancelled" message
  5. Click "Save Changes" button and accept the confirmation
  6. Verify "Save confirmed" message appears in result log
  7. Click "Reset Form" button and dismiss the confirmation
  8. Verify "Reset cancelled" message appears in result log

Expected Results: Accept should trigger positive action, dismiss should cancel action, appropriate messages logged

Test Data: N/A (confirmations use predefined messages)

Selenium Test Code - Confirm Dialogs

@Test
public void confirmDialogsTest() throws InterruptedException {
    // Finding confirm dialog buttons
    WebElement deleteButton = driver.findElement(By.xpath("//button[@onclick='showDeleteConfirm()']"));
    WebElement navigationButton = driver.findElement(By.xpath("//button[@onclick='showNavigationConfirm()']"));
    WebElement saveButton = driver.findElement(By.xpath("//button[@onclick='showSaveConfirm()']"));
    WebElement resetButton = driver.findElement(By.xpath("//button[@onclick='showResetConfirm()']"));
    WebElement confirmResult = driver.findElement(By.id("confirm-result"));
    WebDriverWait wait = new WebDriverWait(driver, Duration.ofSeconds(10));
    // Test Delete Confirm - Accept
    deleteButton.click();
    Alert deleteAlert = wait.until(ExpectedConditions.alertIsPresent());
    String deleteAlertText = deleteAlert.getText();
    Assertions.assertEquals("Are you sure you want to delete this item?", deleteAlertText);
    deleteAlert.accept();
    Thread.sleep(1000);
    String resultAfterAccept = confirmResult.getText();
    Assertions.assertTrue(resultAfterAccept.contains("Delete confirmed"));
    // Test Navigation Confirm - Dismiss
    navigationButton.click();
    Alert navigationAlert = wait.until(ExpectedConditions.alertIsPresent());
    String navigationAlertText = navigationAlert.getText();
    Assertions.assertTrue(navigationAlertText.contains("unsaved changes"));
    navigationAlert.dismiss();
    Thread.sleep(1000);
    String resultAfterDismiss = confirmResult.getText();
    Assertions.assertTrue(resultAfterDismiss.contains("Navigation cancelled"));
    // Test Save Confirm - Accept
    saveButton.click();
    Alert saveAlert = wait.until(ExpectedConditions.alertIsPresent());
    saveAlert.accept();
    Thread.sleep(1000);
    String saveResult = confirmResult.getText();
    Assertions.assertTrue(saveResult.contains("Save confirmed"));
    // Test Reset Confirm - Dismiss
    resetButton.click();
    Alert resetAlert = wait.until(ExpectedConditions.alertIsPresent());
    resetAlert.dismiss();
    Thread.sleep(1000);
    String resetResult = confirmResult.getText();
    Assertions.assertTrue(resetResult.contains("Reset cancelled"));
}

Prompt Dialogs

These dialogs collect user input through a text field.

Test Case: Prompt Dialogs

Test ID: TC_PROMPT_001

Objective: Verify prompt dialogs can accept user input, be cancelled, and handle different data types

Test Steps:

  1. Click "Enter Name" button and verify prompt text: "Please enter your name:"
  2. Enter "John Doe" in prompt and accept, verify "Name entered: John Doe" appears
  3. Click "Enter Age" button and verify prompt text: "Please enter your age:"
  4. Enter "25" in prompt and accept, verify "Age entered: 25" appears
  5. Click "Enter Email" button and dismiss without entering text
  6. Verify "Email prompt cancelled" appears in result log
  7. Click "Enter Password" button and enter "mypassword123"
  8. Accept and verify "Password entered (hidden for security)" appears

Expected Results: Entered text should be captured, cancelled prompts should be logged, passwords hidden

Test Data: Name: "John Doe", Age: "25", Password: "mypassword123"

Selenium Test Code - Prompt Dialogs

@Test
public void promptDialogsTest() throws InterruptedException {
    // Finding prompt dialog buttons
    WebElement nameButton = driver.findElement(By.xpath("//button[@onclick='showNamePrompt()']"));
    WebElement ageButton = driver.findElement(By.xpath("//button[@onclick='showAgePrompt()']"));
    WebElement emailButton = driver.findElement(By.xpath("//button[@onclick='showEmailPrompt()']"));
    WebElement passwordButton = driver.findElement(By.xpath("//button[@onclick='showPasswordPrompt()']"));
    WebElement promptResult = driver.findElement(By.id("prompt-result"));
    WebDriverWait wait = new WebDriverWait(driver, Duration.ofSeconds(10));
    // Test Name Prompt - Enter text and accept
    nameButton.click();
    Alert namePrompt = wait.until(ExpectedConditions.alertIsPresent());
    String namePromptText = namePrompt.getText();
    Assertions.assertEquals("Please enter your name:", namePromptText);
    
    namePrompt.sendKeys("John Doe");
    namePrompt.accept();
    Thread.sleep(1000);
    String nameResult = promptResult.getText();
    Assertions.assertTrue(nameResult.contains("Name entered: John Doe"));
    // Test Age Prompt - Enter number and accept
    ageButton.click();
    Alert agePrompt = wait.until(ExpectedConditions.alertIsPresent());
    String agePromptText = agePrompt.getText();
    Assertions.assertEquals("Please enter your age:", agePromptText);
    
    agePrompt.sendKeys("25");
    agePrompt.accept();
    Thread.sleep(1000);
    String ageResult = promptResult.getText();
    Assertions.assertTrue(ageResult.contains("Age entered: 25"));
    // Test Email Prompt - Cancel without entering text
    emailButton.click();
    Alert emailPrompt = wait.until(ExpectedConditions.alertIsPresent());
    emailPrompt.dismiss();
    Thread.sleep(1000);
    String emailResult = promptResult.getText();
    Assertions.assertTrue(emailResult.contains("Email prompt cancelled"));
    // Test Password Prompt - Enter text and accept
    passwordButton.click();
    Alert passwordPrompt = wait.until(ExpectedConditions.alertIsPresent());
    passwordPrompt.sendKeys("mypassword123");
    passwordPrompt.accept();
    Thread.sleep(1000);
    String passwordResult = promptResult.getText();
    Assertions.assertTrue(passwordResult.contains("Password entered"));
}

Form with Alert Integration

This form demonstrates alerts in real-world scenarios.

Test Case: Form with Alert Integration

Test ID: TC_FORM_ALERT_001

Objective: Verify form submission with alert confirmations and form clearing operations

Test Steps:

  1. Fill username field with "testuser"
  2. Fill email field with "test@example.com"
  3. Select "IT" from department dropdown
  4. Fill comments field with "This is a test comment"
  5. Click "Submit Form" button and verify confirmation dialog contains all form data
  6. Accept the confirmation and verify success alert appears
  7. Verify form is cleared after successful submission
  8. Fill form again and click "Clear Form" button
  9. Accept clear confirmation and verify form fields are empty
  10. Fill form and click "Reset Form", then dismiss confirmation
  11. Verify form data is preserved when reset is cancelled

Expected Results: Form submission should show confirmation with data, clearing should work, reset cancellation should preserve data

Test Data: Username: "testuser", Email: "test@example.com", Department: "IT", Comments: "This is a test comment"

Selenium Test Code - Form with Alerts

@Test
public void formWithAlertsTest() throws InterruptedException {
    // Finding form elements
    WebElement usernameField = driver.findElement(By.id("username"));
    WebElement emailField = driver.findElement(By.id("email"));
    WebElement departmentSelect = driver.findElement(By.id("department"));
    WebElement commentsField = driver.findElement(By.id("comments"));
    WebElement submitButton = driver.findElement(By.xpath("//button[@type='submit']"));
    WebElement clearButton = driver.findElement(By.xpath("//button[@onclick='clearForm()']"));
    WebElement resetButton = driver.findElement(By.xpath("//button[@onclick='confirmFormReset()']"));
    WebElement formResult = driver.findElement(By.id("form-result"));
    WebDriverWait wait = new WebDriverWait(driver, Duration.ofSeconds(10));
    // Fill form with test data
    usernameField.sendKeys("testuser");
    emailField.sendKeys("test@example.com");
    
    Select departmentDropdown = new Select(departmentSelect);
    departmentDropdown.selectByValue("IT");
    
    commentsField.sendKeys("This is a test comment");
    // Test form submission with confirm dialog
    submitButton.click();
    
    Alert submitConfirm = wait.until(ExpectedConditions.alertIsPresent());
    String confirmText = submitConfirm.getText();
    
    // Verify confirmation dialog contains form data
    Assertions.assertTrue(confirmText.contains("testuser"));
    Assertions.assertTrue(confirmText.contains("test@example.com"));
    Assertions.assertTrue(confirmText.contains("IT"));
    Assertions.assertTrue(confirmText.contains("This is a test comment"));
    
    submitConfirm.accept();
    
    // Verify success alert
    Alert successAlert = wait.until(ExpectedConditions.alertIsPresent());
    String successText = successAlert.getText();
    Assertions.assertEquals("Form submitted successfully!", successText);
    successAlert.accept();
    
    Thread.sleep(1000);
    String resultText = formResult.getText();
    Assertions.assertTrue(resultText.contains("Form submitted successfully"));
}
